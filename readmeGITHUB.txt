### Movie Recommendation System

This Python script is a content-based movie recommendation system that suggests movies similar to a user's favorite movie. It uses the cosine similarity metric to find the most similar movies based on various features such as genres, keywords, taglines, cast, director, and overview. The system takes the title of the user's favorite movie as input and returns the top 10 most similar movies.
